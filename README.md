# 🛒 电商图片生成小程序 - 图小智

一款基于 AI 的电商图片生成微信小程序，支持商品主图、背景图、营销海报等图片的智能生成。

---

## 📖 项目简介

**图小智** 是一款面向电商从业者的 AI 图片生成工具，帮助商家快速生成高质量商品主图和营销素材，提升店铺运营效率。

## ✨ 功能特色

### 🎨 核心功能
- **AI 商品主图生成** — 输入商品信息，AI 自动生成精美主图
- **背景替换** — 智能识别商品，一键替换纯色/场景背景
- **营销海报生成** — 输入营销文案，生成吸引眼球的推广海报
- **灵感广场** — 浏览和学习其他用户的优秀作品

### 💰 增值服务
- **余额充值** — 按需充值，支持多种面额
- **积分兑换** — 使用积分兑换更多生成次数
- **邀请奖励** — 邀请好友，双方获得奖励积分
- **转账功能** — 账户间积分互转

### 👥 用户中心
- **历史记录** — 回顾过往生成记录
- **充值明细** — 查看充值和消费明细
- **余额日志** — 完整的账户变动记录

## 🚀 使用说明

### 微信开发者工具
1. 克隆本仓库到本地
2. 打开 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
3. 导入本项目目录
4. 修改 `utils/config.js` 中的 API 配置
5. 编译预览

### 部署后端
本项目需要搭配后端 API 使用，请确保：
- 配置正确的 API 地址
- 完成用户认证和支付接口对接
- 接入 AI 图片生成服务
